# 第一篇 Markdown 博客文章

新的博客模块已经支持直接读取 Markdown 文件内容，你以后可以继续沿用这种方式写作。

## 目前支持的内容

- 一级到四级标题
- 无序列表和有序列表
- 行内代码，例如 `SpringBoot`
- 代码块
- 引用块
- 链接和图片

## 一个简单示例

```java
public class HelloBlog {
    public static void main(String[] args) {
        System.out.println("hello markdown");
    }
}
```

> 你可以把这里当成技术笔记的起点，后面逐步积累成自己的知识库。

如果你想继续写下一篇文章，只需要新增一个 `.md` 文件，然后重新生成博客索引即可。
