# 为什么我想把个人主页重构成博客型站点

之前的主页更像一个静态展示页，适合放项目，但不太适合持续更新内容。

## 我这次主要想解决的问题

1. 整个站点统一为中文表达，信息更集中。
2. 首页不再只是项目堆叠，而是拆成个人介绍、个人经历、项目经历和博客入口。
3. 博客内容改为 Markdown 管理，方便长期维护。

## 后续打算

- 把技术笔记、项目复盘、随笔记录继续拆成独立分类。
- 每完成一个项目，就补一篇复盘文章。
- 让主页负责展示结构，让博客负责沉淀内容。

这样一来，这个站点就不只是“给别人看”的页面，也会变成我自己的长期记录空间。
