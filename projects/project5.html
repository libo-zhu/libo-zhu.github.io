项目简介

本平台整合历史气象数据与实时气象数据，提供实时天气查询、历史数据回溯、生活指数以及灾害预警等功能。系统前端采用 Vue3 构建，后端基于 SpringBoot 与 MySQL 实现数据管理。

技术栈

前端：Vue3 / HTML5 / CSS3 / JavaScript
后端：SpringBoot / MySQL
图表：Echarts

主要职责

  • 设计前端页面，实现响应式布局与交互特效。
  • 爬取气象数据，构建并优化数据存储方案。
  • 整合高德地图与第三方 API 获取实时天气与生活指数数据。
  • 利用 Echarts 实现多种图表展示，包括折线图、饼图、热力图等。
