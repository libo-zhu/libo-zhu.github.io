项目简介

构建一个多源气象（历史数据+实时数据）数据可视化平台。包含实时天气查询、历史气象数据查询、生活 指数、灾害预警4 个模型功能。前端使用vue3,后端使用SpringBoot,MySQL 用于数据持久化，以及Echarts 用于图表的 绘制。

技术栈

前端：Vue3 / HTML5 / CSS3 / JavaScript
后端：SpringBoot / MySQL
图表：Echarts

主要职责

  • 对中国气象局进行数据的爬虫，得到30 万条历史数据，包括城市、日期、最高气温、最低气温、风向信息；
  • 使用高德地图、易客天气API，获取实时的天气预报、生活指数、灾害预警，并使用Echarts 实现 折线图、饼图、热力图、雷达图等多种可视化方式；
  • 使用SpringBoot+MySQL，实现登录注册功能，以及历史数据的持久化与检索。
